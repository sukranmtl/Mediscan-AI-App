# 💊 Mediscan AI: Büyüklerimizin Sağlık Rehberi

**Mediscan AI**, yaşlı bireylerin karmaşık ilaç prospektüslerini anlamalarını kolaylaştırmak için geliştirilmiş, yapay zeka destekli bir sağlık asistanıdır. Google Gemini API kullanılarak hazırlanan bu proje, biyomedikal mühendisliği bakış açısıyla erişilebilirlik sorunlarına çözüm sunar.

## 🌟 Öne Çıkan Özellikler
- **Yaşlı Dostu Tasarım:** Dev yazı tipleri, yüksek kontrastlı renkler ve sade arayüz.
- **Yapay Zeka ile Sadeleştirme:** Tıbbi terimleri (endikasyon, kontrendikasyon vb.) günlük dile çevirir.
- **Kategorize Edilmiş Bilgi:** "Nasıl Kullanılır?", "Dikkat Edilecekler" ve "Yan Etkiler" başlıklarıyla net özet sunar.
- **Güvenli API Yönetimi:** Kullanıcı tabanlı API anahtarı girişi ile güvenlik sağlar.

## 🛠️ Teknik Detaylar
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide-react
- **AI Model:** Google Gemini 1.5 Flash (v1/v1beta)
- **Deployment:** Vercel / Netlify

## 🚀 Kurulum ve Çalıştırma
1. Bu depoyu klonlayın.
2. `npm install` ile bağımlılıkları yükleyin.
3. `npm run dev` ile yerel sunucuyu başlatın.
4. Tarayıcıda uygulamayı açıp kendi Gemini API anahtarınızı girerek kullanmaya başlayın.

## 🎓 Geliştirici Hakkında
Bu proje, bir Biyomedikal Mühendisliği öğrencisi tarafından, yaşlıların sağlık okuryazarlığını artırmak amacıyla bir Buildathon kapsamında geliştirilmiştir.
